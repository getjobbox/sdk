# JobBox Python SDK

**Status:** Planned  
**Target package:** `getjobbox` or `getjobbox-sdk` (PyPI)

This directory will hold the Python partner SDK. Implementation must follow [`../STANDARD.md`](../STANDARD.md).

Until this package is published, use the HTTP API directly with header `X-JobBox-Api-Key`, or the Node SDK at [`../node`](../node).
